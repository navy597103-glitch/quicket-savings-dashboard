# QUICKET Lifecycle Savings Dashboard

This repository contains a Vite + React + Tailwind CSS single-page app version of the QUICKET Lifecycle Savings Dashboard.

Features:
- React + Vite
- Tailwind CSS
- Recharts for charts
- Lucide icons

Local dev:
1. npm install
2. npm run dev

Build for production:
- npm run build

Deploy to Vercel:
- Link the repository in Vercel and set the framework to "Other" or "Vite". Build command: `npm run build` and Output Directory: `dist`.
